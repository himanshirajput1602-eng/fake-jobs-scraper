# Fake Jobs Web Scraper

A Python web scraper that collects job listings from the Fake Python Jobs website.

## Project URL
https://realpython.github.io/fake-jobs/
https://roadmap.sh/projects/job-listings-scraper

## What it does
Scrapes job title, company name, location, and job detail URL from the site
and saves the results to a `jobs.csv` file.

## Requirements
pip install requests beautifulsoup4

## Usage
python job_scraper.py
